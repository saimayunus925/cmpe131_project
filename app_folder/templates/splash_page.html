Tasksent

A Little About Us... (spash page)

Tasksent is a scheuduling site that allows account bearing users to establish meeting times and durations. Guest users may access the site and view these meetings or avalibility timeframes.

Users who sign up on the site are allowed to dictate time avalibility wthin an hourly range spanning from 9:00am to 10:00pm. Meeting duration, or availibility windows, can then be chosen from the following options: 1 quarter hour, 1 half hour, or 1 full hour.

Users also have the option of sending email notificaions to guests who have selected a specific avalibility time.

To check out our website, click here, or select "Home" from the navigation bar. Contine below to read about the site's functions.

Site Functions:

Navagation Bar:

The navagation bar can be accessed from the top of the site, below the browser search bar.
This bar allows visitors to swap between the "Home", "About", "Login", and "Create Account" pages.
Account bearing users also have the additional links to "Logout", "Meetings", and "Settings".

Create Account page:

Takes inputs "Email", "Username", and "Password".
This page can be accessed from either the navigation bar, via a redirection
from the login page, or by appending "/create-account" to the site link.

Delete Account page:

Takes varification inputs "Username", and "Password".
This page can only be accessed from the settings page via the navigation bar
when a user is logged in. Alternatively, a user can append "/delete-account" to the site link.

Login Page and Logout Functionality:

Login: Takes inputs "Username", and "Password" and has a checkable option "remember_me".
Login can be accessed from either the navigation bar, or by appending "/login" to the site link.

Logout: Allows a user to exit account on from the site.
Logout can be accessed via the navigation bar,
or a force logout action can be committed by appending "/logout" to the site link.

Settings page:

Prompts user to: "change availability", "select meeting length, and "send email confirmation".
Also allows users to redirect to delete account page.
This page can ONLY be accessed when a user is logged in and can be seen from either the navigation bar,
or by appending "/settings" to the site link.

Meetings page:

Display's a logged-in user's current meetings.
This page can ONLY be accessed when a user is logged in and can be seen from either the navigation bar,
or by appending "/meetings" to the site link.
